【概要】

　C++ で MIDI IN メッセージを受信するサンプルです。デバイスを確認し
（複数あれば選択して）開きます。その後データを受信して表示します。
　実行ファイルは MIDI の受信を試してモニタしてみたいという場合にも使
えるかもしれません。


【最近の変更点】
Ver 1.1
　・メッセージの内容を解析して表示するようにした
Ver 1.0
　・初出

【動作環境】

・Microsoft Windows XP HomeEdition
・ひとつ以上の正常に接続された MIDI 機器
　その他の環境では未確認ですが、Windows 系ならたいていは作動すると思
います。MIDI の種類は関係ないはずです。
